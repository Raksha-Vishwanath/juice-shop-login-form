# Juice Shop Inspired Login Form

## Overview
This project is a simple login for OWASP Juice Shop. It demonstrates both client-side and server-side validation to prevent common input errors and improve security.

## Technologies Used
- HTML
- CSS
- JavaScript (Frontend)
- Node.js (Backend)

## Features
- Email and password input fields
- Client-side validation:
  - Checks for empty fields
  - Ensures email contains "@"
  - Ensures password is at least 8 characters
- Server-side validation:
  - Validates input again for security
  - Returns appropriate messages
